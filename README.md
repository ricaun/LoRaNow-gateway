# LoRaNow-gateway
LoRaNow gateway

## Dependences Library

* [LoRaNow][10]
* [CayenneLPPDecode][11] 
* [ArduinoJson][16]
* [ESPAsyncWebServer][12]
* ESP8266 - [ESPAsyncTCP][13] or ESP32 - [AsyncTCP][14]
* [ESPAsyncWiFiManager][15]
